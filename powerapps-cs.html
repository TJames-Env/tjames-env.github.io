HOW TO USE

Overview

Power Apps canvas app developer reference — formulas, delegation rules, UI patterns, data source operations, performance best practices, and reusable component design.

Variable Bar

  • app name — your canvas app display name
  • sharepoint site — SharePoint site URL for data sources
  • dataverse env — Dataverse environment URL
  • admin email — App owner / admin email
  • app id — Canvas app GUID (from maker portal URL)

Navigation

  • Left sidebar — filter by category
  • Tag pills — filter by type
  • Search — find by formula name or keyword
  • Click any card — full pattern in right panel
  • Copy  |  Copy+Enter for quick paste
