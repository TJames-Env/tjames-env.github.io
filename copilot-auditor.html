//  How to Export an Agent
// Method 1 — Export from Copilot Studio Portal 1. Go to copilotstudio.microsoft.com
2. Open the agent you want to audit
3. Click Settings → Export → Export as solution (.zip)
4. Unzip → find the bot definition JSON inside the solution
5. Paste the contents here
// Method 2 — Power Platform CLI
# Install CLI
npm install -g @microsoft/powerplatform-cli
# Export agent solution
pac solution export --path agent.zip --name AgentName
# Unzip and find bot JSON
unzip agent.zip -d ./agent-export
