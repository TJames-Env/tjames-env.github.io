//  How to Export a Flow
// Method 1 — Export from Power Automate Portal 1. Go to make.powerautomate.com
2. Open the flow you want to analyse
3. Click ⋮ (three dots) → Export → Package (.zip)
4. Unzip the package → inside is a definition.json file
5. Paste the contents of that file here
