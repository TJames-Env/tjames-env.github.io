HOW TO USE

Overview

Recon Arsenal is an all-in-one enumeration cheatsheet with live variable substitution. Fill in your target variables in the top bar and every command updates automatically. Click any command to copy it.

Variable Bar

Every {VARIABLE} in commands is automatically replaced as you type.

VariableDescriptionExample
{IP}Target IP address10.10.14.166
{DOMAIN}Target domain namevesper.local
{USER}Usernamejsmith
{PASS}PasswordPassword123!
{PORT}Target port80
{LHOST}Your attacker IP10.10.14.1
{LPORT}Your listener port4444
{WORDLIST}Wordlist path/usr/share/wordlists/dirb/common.txt
{DC}Domain controller IP or hostname10.10.14.166
{HASH}NTLM hash (NT:LM or full)aad3b435b51...

Navigating Categories

  • Use the left sidebar to filter by technique category
  • Use the tag filters in the header bar to filter by tag
  • Use the search box to find any command by name, description, or tag
  • Click All to see every cheat across all categories

Using Commands

  • Click any card to open the full command in the right panel
  • Click a command block or Copy to copy to clipboard
  • Copy+Enter copies the command with a newline (paste and run)
  • Commands show multiple variants where available

Keyboard Shortcuts

  • Ctrl + K  — Focus search
  • Esc  — Clear selection / close panel
  • ↑ ↓  — Navigate cards when search is focused
  • ?  — Open this help modal

Tips

  • Variables not yet filled show as {PLACEHOLDER} — fill the var bar first
  • The AI / LLM Enum category has Claude-powered prompt templates
  • All data stays in your browser — nothing is sent anywhere
eusebio@it-sec:~$
recon-arsenal Local-only Runs entirely in your browser — no data is uploaded or sent anywhere
